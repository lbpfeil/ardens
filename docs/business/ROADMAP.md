# Arden — Roadmap de Negócio

**Última atualização:** 2026-01-26

## Visão Geral
Lançar e validar o Arden no mercado brasileiro de construção civil (PMEs com PBQP-H) em 2026, atingindo 10-20 clientes pagantes até o final do ano.

## Fases

### v1.0: MVP (Fundação)
**Status:** Concluída (2026-01-24)
**Objetivo:** Validar arquitetura e fluxos básicos de cadastro.
- [x] Portal web (obras, biblioteca FVS)
- [x] Dashboard básico
- [x] Estrutura de dados Multi-tenant

### v1.1: Verificações Web
**Status:** Em andamento
**Objetivo:** Permitir o ciclo completo de qualidade sem depender do mobile (útil para admins e testes).
- [ ] Execução de FVS pelo navegador
- [ ] Upload de fotos via web

### Beta Fechado (Março 2026)
**Status:** Planejada
**Objetivo:** Validar produto em ambiente real com parceiros.
- [ ] Onboarding de 1-2 construtoras parceiras (uso gratuito)
- [ ] Coleta de feedback intensiva

### Fase 2: Mobile & Expansão
**Status:** Planejada
**Objetivo:** Entregar a experiência "de campo" real.
- [ ] App Android (Offline-first)
- [ ] App iOS
- [ ] Condições de Início (CI)

### Futuro / Exploratório
**Status:** Em análise
- **Internacionalização:** Estudo de viabilidade para Portugal (ISO 9001).
- **Automação:** Relatórios preditivos e IA.
- **Marketing/SEO:** Páginas de comparação (Arden vs. Excel, Arden vs. AutoDOC), estratégia de conteúdo para "FVS digital" e "software PBQP-H". (Recomendações da pesquisa competitiva — em análise)
